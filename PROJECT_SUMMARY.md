## 🎨 핵심 기능

### 1. 색상 테스트
- 180개 색상 무작위 셔플
- 전체 화면 색상 표시
- 마우스/키보드/터치 지원
- 진행상황 실시간 표시

### 2. 진단 알고리즘
```
입력: 사용자가 좋아한 색상들
처리:
  1. 평균 명도(Lightness) 계산
  2. 평균 채도(Saturation) 계산
  3. 평균 색상각도(Hue) 계산 (원형 평균)
  4. 온도 판단 (따뜻함/시원함)
  5. 계절 판단 (봄/여름/가을/겨울)
  6. 톤 판단 (라이트/브라이트/뮤티드)
출력: "Season Tone" (예: "Spring Bright")
```

### 3. 결과 분석
- 진단된 계절 색상 타입
- 사용자가 좋아한 모든 색상
- 추천 색상 (상위 6개)
- 피해야 할 색상 (반대 타입 상위 6개)
- 색상 분석 설명

## 💡 주요 구현 사항

### 1. 색상 셔플링
```javascript
const shuffled = [...allColors].sort(() => Math.random() - 0.5);
```
마운트 시 한 번만 실행, 전체 180개 색상 무작위 배열

### 2. 원형 평균 계산
```javascript
const sin = hues.reduce(...) / hues.length;
const cos = hues.reduce(...) / hues.length;
let avg = Math.atan2(sin, cos) * (180 / Math.PI);
```
색상각도의 원형 특성 고려 (0° = 360°)

### 3. 계절 판단
```javascript
// 명도로 라이트/다크 구분
const isLight = avgLightness > 75;

// 채도로 브라이트/뮤티드 구분
const isBright = avgSaturation > 65;

// 색상각도로 따뜻함/시원함 구분
const isWarm = (avgHue < 60 || avgHue > 300);
```

## 🎯 12계절 시스템

### 따뜻한 톤 (봄/가을)
- **봄**: 밝고 선명한 색상
  - Spring Light: 밝은 따뜻한 파스텔
  - Spring Bright: 선명한 따뜻한 색상
  - Spring Muted: 부드러운 따뜻한 색상

- **가을**: 깊고 따뜻한 색상
  - Autumn Light: 밝은 따뜻한 색상
  - Autumn Bright: 깊은 따뜻한 색상
  - Autumn Muted: 부드러운 깊은 따뜻한 색상

### 시원한 톤 (여름/겨울)
- **여름**: 부드럽고 시원한 색상
  - Summer Light: 밝은 시원한 파스텔
  - Summer Bright: 선명한 시원한 색상
  - Summer Muted: 부드러운 시원한 색상

- **겨울**: 깊고 시원한 색상
  - Winter Light: 밝은 시원한 색상
  - Winter Bright: 깊은 시원한 색상
  - Winter Muted: 부드러운 깊은 시원한 색상
