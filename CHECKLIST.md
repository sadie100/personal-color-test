### 포트 이미 사용 중
→ Vite가 자동으로 다음 포트 사용 (5174, 5175 등)

### 빌드 오류
→ npm install -D @tailwindcss/postcss 실행

### 의존성 문제
→ rm -rf node_modules && npm install
